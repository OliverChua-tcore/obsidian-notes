# CSS properties

### `background-size`
| Value         | Purpose                                                |
| ------------- | ------------------------------------------------------ |
| `cover`       | Scales image to cover the entire element (may crop)    |
| `contain`     | Scales image to fit inside the element (may show gaps) |
| `100% 100%`   | Stretches to fill exact dimensions                     |
| `auto`        | Uses image's natural size                              |
| `200px 150px` | Sets explicit width and height                         |
